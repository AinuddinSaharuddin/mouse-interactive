# 🖱️ Mouse Interactive

An interactive website showcasing mouse-driven effects — built with **pure HTML, CSS & JavaScript**. Zero dependencies.

## ✨ Effects

- **Custom Cursor** — Dual-ring cursor with smooth lag, hover & click states
- **Cursor Trail** — 10 fading dots that follow your mouse like a comet tail
- **Hero Parallax** — Background gradients shift in parallax as you move
- **3D Tilt Cards** — Cards tilt in 3D with a spotlight glow tracking your cursor
- **Magnetic Button** — Button physically pulls toward your cursor when nearby
- **Ripple Canvas** — HTML5 canvas ripple rings generated by mouse movement
- **Scrolling Text Band** — Marquee speed changes with mouse horizontal velocity
- **Count-Up Stats** — Numbers animate when scrolled into view
- **Light/Dark Mode** — Theme toggle with smooth transitions

## 🚀 Usage

Just open `mouse-interactive.html` in any modern browser. No build tools, no npm, no setup.

## 🛠️ Built With

- HTML5 Canvas API
- CSS Custom Properties + OKLCH colors
- Vanilla JavaScript (requestAnimationFrame)
- Google Fonts (Instrument Serif + Work Sans)
